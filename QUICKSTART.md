# Quickstart

## Requirements

- Node.js 20+

## Install

```bash
npm install
```

## Run the demo

```bash
npm run demo
```

## Run a custom config

```bash
npm run dev -- run path/to/workflow.yaml
```

## Validate quality gates

```bash
npm run lint
npm run test
npm run typecheck
```

## Expected output

The demo prints a markdown execution report and writes the same content to `artifacts/latest-run.md`.
